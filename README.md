# PedeSampa

Landing page e estrutura inicial do projeto **PedeSampa**, uma iniciativa voltada para criação de soluções digitais para pequenos negócios de delivery em São Paulo.

O projeto nasce com foco em restaurantes, hamburguerias, pizzarias, marmitarias e negócios locais que querem vender melhor pelo próprio canal, organizar pedidos e reduzir a dependência exclusiva dos grandes aplicativos de entrega.

## Sobre o projeto

O PedeSampa foi criado para apresentar uma proposta simples e direta: ajudar pequenos negócios a terem uma presença digital mais profissional, com páginas modernas, comunicação clara e estrutura preparada para evoluir para apps de delivery, cardápios digitais e automações.

A ideia é transformar tecnologia, IA e desenvolvimento web em soluções acessíveis para empreendedores locais que precisam vender mais, atender melhor e organizar sua operação.

Este repositório contém a landing page inicial do projeto.

## Objetivo

O objetivo desta primeira versão é validar a comunicação, posicionamento e proposta comercial do PedeSampa.

A landing page apresenta a ideia do serviço, o público-alvo e a proposta de valor para negócios de delivery que desejam ter um canal próprio de vendas.

## Público-alvo

O projeto é voltado para pequenos negócios como:

- Restaurantes
- Hamburguerias
- Pizzarias
- Marmitarias
- Lanchonetes
- Açaíterias
- Negócios locais de delivery
- Empreendedores que vendem pelo WhatsApp ou redes sociais

## Proposta de valor

O PedeSampa busca ajudar negócios locais a:

- Criar uma presença digital própria
- Apresentar produtos de forma mais profissional
- Receber pedidos com mais organização
- Melhorar a experiência do cliente
- Usar o WhatsApp como canal comercial de forma mais estratégica
- Reduzir a dependência de marketplaces de delivery
- Testar soluções digitais com baixo custo inicial
- Evoluir para automações, cardápios inteligentes e apps personalizados
